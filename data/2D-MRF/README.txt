Instances corresponding to inference problems problems of the form c'x+x'Qx, in the following format:
Row 1: dimension (n)
Row 2: constant to add to the objective (=\|y\|_2^2)
Row 3: n-dimensional vector with the support of the true underlying signal (=\|y\|_0)
Row 4: n-dimensional vector with the true underlying signal (y)
Row 5: vector c
rows 6-(n+6): matrix Q
